🧩 zsteg 输出结构快速理解

zsteg 会尝试从 PNG 的像素数据中，以不同方式提取隐藏信息，例如：

  • b1 / b2 / b4：每像素使用 1、2、4 bit 的最低位（LSB）或最高位（MSB）

  • r / g / b / rgb / bgr：使用红、绿、蓝通道或组合

  • lsb / msb：最低有效位 / 最高有效位

  • xy：按像素顺序扫描

  • text / file：尝试解析为文本或文件头

你给出的输出中，每一行都是一种“提取方式 + 解析结果”。
